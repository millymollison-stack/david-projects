# Communication Preferences

## How to Respond to Messages

### Tasks from David
- ALWAYS respond with "OK" immediately when receiving a task via messenger
- Confirm understanding of the task
- Give time estimate if possible

### URLs
- NEVER use ** or * around URLs
- Just paste the raw URL
- Example: https://example.com (not **https://example.com**)

### Updates
- Confirm when tasks are done
- Provide live links without formatting

## Permanent Rules

1. Say "OK" when receiving any task
2. No asterisks around URLs
3. Confirm URLs work before sharing
4. Update the person immediately when tasks complete
